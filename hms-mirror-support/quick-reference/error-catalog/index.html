    Error Message Catalog¶

    When a customer reports an error, find the matching entry here to identify cause and next steps.

    Server / Startup Errors¶

    Error Message Cause Action
    Address already in use: 8090 Port 8090 in use by another process Check with lsof -i :8090; stop other process or change port
    Cannot open RocksDB — resource temporarily unavailable Another HMS-Mirror instance running, or stale LOCK file Verify no other instance; remove ~/.hms-mirror/data/LOCK if confirmed
    RocksDB: Corruption detected Database integrity failure L3: backup + recovery
    Keystore not found at path Security keystore file missing Verify ~/.hms-mirror/security/hms-mirror.p12 exists
    Failed to load keystore — UnrecoverableKeyException Wrong keystore password in web.properties Fix hms-mirror.keystore.password
    UnsupportedClassVersionError Wrong Java version Upgrade to Java 17+
    Could not find or load main class Corrupted JAR or wrong startup command Re-download/rebuild the JAR
    Port 8090 was already in use Same as Address already in use Same resolution
    OutOfMemoryError at startup Heap too small Increase JVM heap with -Xmx

    Authentication Errors¶

    Error Message Cause Action
    Invalid credentials Wrong username or password Verify credentials; reset if needed
    Account is locked 5+ failed login attempts Admin unlocks via UI or API
    Too many login attempts (HTTP 429) Rate limit hit Wait 1 minute
    JWT token has expired Access token older than 15 minutes Use refresh token or log in again
    JWT signature does not match JWT secret changed since token issued Log in again
    Unauthorized (HTTP 401) No token provided Add Authorization: Bearer <token> header
    Forbidden (HTTP 403) Token valid but role insufficient User lacks required role (ADMIN vs USER)
    Authentication failed for user: <name> (in log) Credential verification failed Check password, LDAP connectivity
    LDAP connection refused LDAP server not reachable Check LDAP URL configuration

    Connection / JDBC Errors¶

    Error Message Cause Action
    Connection refused Service not running on target host/port Verify HiveServer2 is running
    No suitable driver found for jdbc:hive2:// Missing JDBC driver JAR Place Hive JDBC JAR in ~/.hms-mirror/aux_libs/
    GSS initiate failed No valid Kerberos TGT Run kinit
    Clock skew too great Time difference > 5 min with KDC Sync NTP on HMS-Mirror host
    KDC unreachable Cannot reach KDC on port 88 Check network/firewall to KDC
    Server not found in Kerberos database Wrong principal in JDBC URI Verify ;principal= in JDBC URI
    SSL handshake failed Certificate trust issue Import cluster CA into trust store
    PKIX path building failed Same as SSL handshake Import CA cert
    Access denied for user Missing Hive/HDFS permissions Grant permissions to migration user
    HTTP 401 Unauthorized (Knox) Wrong Knox credentials Verify Knox username/password
    HTTP 403 Forbidden (Knox) Knox authorization denied Check Knox topology permissions

    Migration / Job Errors¶

    Error Message Cause Action
    Table already exists Target table exists from previous run Enable transfer.overwrite or drop table
    Database does not exist Target database not created Create database on target cluster
    Cannot export transactional table ACID table in EXPORT strategy Enable acid.downgradeAcid: true
    OutOfMemoryError during job Table too large for SQL strategy Switch to EXPORT_IMPORT strategy
    Query timed out Long-running INSERT Switch strategy or increase transfer.queryTimeout
    Too many dynamic partitions Partition limit on target Set hive.exec.max.dynamic.partitions override
    Dynamic partition strict mode Target Hive rejects dynamic partitions Set hive.exec.dynamic.partition.mode=nonstrict
    Permission denied on export path Can't write to HDFS staging Grant write permission on staging path
    No space left on device (EXPORT) Insufficient staging disk space Use different staging path
    Type mismatch / Type conversion Incompatible column types Review DDL at source vs target

    RocksDB Errors¶

    Error Message Cause Action
    Cannot open database Lock held or corruption L3: Check lock file, then escalate
    IO error Filesystem or disk issue Check disk health and space
    No space left on device RocksDB data dir full Free disk space
    Corruption detected Database integrity failure L3: Backup and recovery
    Permission denied: rocksdb Wrong ownership/permissions chown -R $USER ~/.hms-mirror/data/

    HTTP Error Codes¶

    Code Meaning Common Causes
    400 Bad Request Malformed JSON, missing required fields
    401 Unauthorized No token, expired token
    403 Forbidden Token valid, role insufficient
    404 Not Found Wrong URL (missing /hms-mirror prefix), resource deleted
    409 Conflict Duplicate resource (username, connection name)
    429 Too Many Requests Login rate limit
    500 Internal Server Error Unhandled exception — check server log
    503 Service Unavailable Server starting up or overloaded

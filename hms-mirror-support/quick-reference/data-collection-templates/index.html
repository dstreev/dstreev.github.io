## Template E: Customer Information Request

Use this when you need more info from the customer before you can diagnose:
    Hello,

    Thank you for contacting support regarding your HMS-Mirror issue.

    To help us investigate, please provide the following:

    1. HMS-Mirror version — found in the JAR filename or from running: 

      java -jar hmsm-server-*.jar --version

    2. Java version: 

      java -version

    3. Last 200 lines of the application log: 

      tail -200 ~/.hms-mirror/logs/hms-mirror.log

    4. Health check result: 

      curl -s http://localhost:8090/hms-mirror/api/v1/system/health

    5. Exact error message — please copy the complete text, not a screenshot if possible.

    6. What you were trying to do when the error occurred.

    Please note: if any configuration files contain passwords, replace them with <REDACTED> before sending.

    Thank you, HMS-Mirror Support ```

    Engineering Reference¶

    Audience: Platform engineers, developers, and engineering support staff working with HMS-Mirror source code or production deployments requiring code-level investigation.

    Prerequisites: Full L1/L2/L3 knowledge, Java and Spring Boot proficiency, familiarity with the Hadoop/Hive ecosystem.

    What This Section Covers¶

    • Building HMS-Mirror from source
    • Internal architecture: modules, packages, key classes
    • RocksDB column family schema — what is stored where
    • Security architecture: JWT internals, keystore design, auth flows
    • Known issues, workarounds, and technical debt notes

    Quick Links for Engineering Work¶

    Build¶

    # Full build
mvn clean package -DskipTests

# Server only (faster)
mvn clean package -pl server -am -DskipTests

# Run tests
mvn test -pl server

# Version management
./update_version.sh 4.1.5.9

    Area Location
    REST Controllers server/src/main/java/.../web/controller/api/v1/

    Security / JWT server/src/main/java/.../security/

    Domain models common/src/main/java/.../domain/

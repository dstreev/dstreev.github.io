Running LLAP on Compute Only Nodes

Oct 15, 2019 1 min.

To run LLAP and/or Hive jobs on Compute Only Nodes you need to override a Hive Configuration default setting. Otherwise, queries will fail a majority of the time. The hive configuration value hive.merge.nway.joins is true by default. But in Ambari 2.7.3+ (HDP 3+), it overrides the default setting and configures this to false. When this is left at the default true, intermittent failures of queries will happen on compute only nodes.

The Power of Hive JDBC Federation

Oct 15, 2019 1 min.

Hive jdbc-federation is a powerful mechanism to include external sources in your hive ecosystem. Apache Software Foundation has excellent docs detailing this feature referred to as the JDBCStorageHandler . Interesting points about this StorageHandler include: Securing Passwords - Protects passwords using a ‘jceks’ file stored on ‘hdfs’ Partitioning - Could be used as an alternate to SQOOP for importing data to Hive More Hive Metadata! Hive Metadata, previously only available via WebHCat which has been removed, can be retrieved through hive’s sys database.

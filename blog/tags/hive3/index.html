Filtering Compactions in Hive

Oct 15, 2019 3 min.

Compactions can be seen with the standard SQL: SHOW COMPACTIONS; But this method has a couple of problems: No Filtering Timestamps are hard to interpret Until additional functionality is available for this built in function, we can do the following. Add links to the Metastore DB tables and create custom views to review compaction details. Build Out Metadata Elements CREATE DATABASE IF NOT EXISTS custom_sys; DROP TABLE IF EXISTS `custom_sys`.

    Getting Started¶

    Welcome to HMS-Mirror! This guide will help you get up and running quickly with Hive metadata and data migrations.

    What is HMS-Mirror?¶

    HMS-Mirror is your go-to tool for migrating Apache Hive tables between DataLake environments. Whether you're upgrading from HDP to CDP, moving to the cloud, or setting up disaster recovery, HMS-Mirror automates the complex process of synchronizing metadata and data.

    Prerequisites Checklist¶

    Before starting, ensure you have:

    • Java 11 or higher (Java 17 recommended)
    • Network access to your Hive clusters (HiveServer2 ports)
    • Valid credentials for cluster authentication
    • Kerberos ticket (if your clusters use Kerberos)

    Data Strategy Selection¶

    Scenario Strategy
    Schema sync only SCHEMA_ONLY
    Quick read access LINKED
    Small tables (<1GB) SQL
    Large/ACID tables EXPORT_IMPORT
    Mixed workloads HYBRID
    Same cluster, new storage STORAGE_MIGRATION
    Shared storage COMMON
    DDL extraction DUMP
    Format modernization ICEBERG_CONVERSION

    Common Error Quick Fixes¶

    Error Quick Fix
    Connection refused Check host/port/firewall
    Authentication failed Verify credentials, run kinit
    Table exists Enable overwrite or use new name
    Out of memory Increase -Xmx or split job
    Too many partitions Increase limit or use EXPORT_IMPORT
    ACID errors Enable downgrade or use EXPORT_IMPORT

    Search Tips¶

    Finding Configuration Options¶

    Configuration options follow patterns: - Migration settings: migration.* - Optimization settings: optimization.* - Transfer settings: transfer.* - ACID settings: acid.*

    Finding CLI Options¶

    CLI options use standard conventions: - Short form: -o, -d, -e - Long form: --output, --database, --execute - Environment: HMS_MIRROR_*

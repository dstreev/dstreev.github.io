    Configuration Reference¶

    This reference documents all HMS-Mirror configuration options.

    Application Properties¶

    Configure HMS-Mirror server settings in application.properties or application.yml.

    Server Configuration¶

    Property Description Default
    server.port HTTP port 8090
    server.servlet.context-path URL context path /hms-mirror 
    server.port=8090
server.servlet.context-path=/hms-mirror

    RocksDB Storage¶

    Property Description Default
    hms-mirror.rocksdb.enabled Enable RocksDB true
    hms-mirror.rocksdb.path Database location ~/.hms-mirror/rocksdb 
    hms-mirror.rocksdb.enabled=true
hms-mirror.rocksdb.path=${user.home}/.hms-mirror/rocksdb

    Output Configuration¶

    Property Description Default
    hms-mirror.config.output-dir Reports directory ~/.hms-mirror/reports
    hms-mirror.config.metastore-drivers-dir JDBC drivers ~/.hms-mirror/aux_libs 
    hms-mirror.config.output-dir=${user.home}/.hms-mirror/reports
hms-mirror.config.metastore-drivers-dir=${user.home}/.hms-mirror/aux_libs

    Logging¶

    Property Description Default
    logging.level.com.cloudera.utils.hms App log level INFO
    logging.file.name Log file path ~/.hms-mirror/logs/hms-mirror.log 
    logging.level.com.cloudera.utils.hms=INFO
logging.file.name=${user.home}/.hms-mirror/logs/hms-mirror.log

    Connection Configuration¶

    HiveServer2 Settings¶

    Property Type Description
    uri String JDBC connection URI
    driverClassName String JDBC driver class
    connectionProperties Map Connection properties
    sessionProperties Map Hive session properties 
    hiveServer2:
  uri: jdbc:hive2://hs2.example.com:10000/default
  driverClassName: org.apache.hive.jdbc.HiveDriver
  connectionProperties:
    transportMode: binary
    ssl: "true"
  sessionProperties:
    hive.exec.parallel: "true"

    Metastore Direct Settings¶

    Property Type Description
    type Enum Database type (MYSQL, POSTGRESQL, ORACLE)
    uri String JDBC connection URI
    username String Database username
    password String Database password 
    metastoreDirect:
  type: MYSQL
  uri: jdbc:mysql://metastore.example.com:3306/hive
  username: hive_user
  password: password

    Partition Discovery¶

    Property Type Description Default
    auto Boolean Auto-discover partitions true 
    partitionDiscovery:
  auto: true

    Migration Configuration¶

    Details Section¶

    Property Type Description
    name String Configuration name
    description String Configuration description

    Migration Section¶

    Property Type Description Default
    copyViews Boolean Migrate view definitions false
    viewsOnly Boolean Only process views false
    evaluatePartitionLocation Boolean Check partition locations false
    tablePropertyFilter String Regex to filter properties null
    resetToDefaultLocation Boolean Reset table locations false 
    migration:
  copyViews: true
  viewsOnly: false
  evaluatePartitionLocation: true
  tablePropertyFilter: "^(?!TRANSLATED).*"
  resetToDefaultLocation: false

    Optimizations Section¶

    Property Type Description Default
    sortDynamicPartitionInserts Boolean Sort before partition insert false
    autoTune Boolean Auto-optimize settings false
    textCompression Boolean Enable text compression false
    compressionCodec String Compression codec null
    buildShadowStatistics Boolean Build stats during migration false
    overrides List Property overrides [] 
    optimizations:
  sortDynamicPartitionInserts: true
  autoTune: true
  textCompression: true
  compressionCodec: ZSTD
  buildShadowStatistics: true
  overrides:
    - key: hive.exec.parallel
      leftValue: "true"
      rightValue: "true"

    Transfer Section¶

    Property Type Description
    warehouse.external String External table warehouse
    warehouse.managed String Managed table warehouse
    commonStorage String Shared storage path
    intermediateStorage String Temp storage for export
    globalLocationMap Map Namespace translations 
    transfer:
  warehouse:
    external: s3a://bucket/warehouse/external
    managed: s3a://bucket/warehouse/managed
  commonStorage: s3a://shared/common
  intermediateStorage: hdfs:///tmp/hms-mirror
  globalLocationMap:
    "hdfs://old-ns": "s3a://new-bucket"

    Storage Migration Section¶

    Property Type Description Default
    translationType Enum RELATIVE or ALIGNED RELATIVE
    dataMovementStrategy Enum SQL, DISTCP, EXPORT_IMPORT, MANUAL SQL
    dataFlow Enum PULL or PUSH PULL 
    transfer:
  storageMigration:
    translationType: ALIGNED
    dataMovementStrategy: DISTCP
    dataFlow: PULL

    ACID Section¶

    Property Type Description Default
    includeAcidTables Boolean Process ACID tables false
    acidOnly Boolean Only process ACID tables false
    downgradeAcid Boolean Convert to external false
    inPlace Boolean In-place migration false
    migrateAcid Boolean Full ACID migration false
    partitionLimit Integer Max partitions 0
    artificialBucketThreshold Integer Min buckets for SQL 0 
    acid:
  includeAcidTables: true
  acidOnly: false
  downgradeAcid: true
  inPlace: false
  migrateAcid: false
  partitionLimit: 500
  artificialBucketThreshold: 2

    Iceberg Section¶

    Property Type Description Default
    enabled Boolean Enable Iceberg conversion false
    conversionMode Enum IN_PLACE or SHADOW SHADOW
    version Integer Iceberg format version 2
    fileFormat Enum PARQUET, ORC, AVRO PARQUET
    tableProperties Map Iceberg table properties {} 
    iceberg:
  enabled: true
  conversionMode: SHADOW
  version: 2
  fileFormat: PARQUET
  tableProperties:
    write.format.default: parquet
    write.parquet.compression-codec: zstd

    Dataset Configuration¶

    Property Type Description
    name String Dataset name
    description String Dataset description
    databases List Database definitions

    Database Definition¶

    Property Type Description
    name String Source database name
    includePattern String Regex for included tables
    excludePattern String Regex for excluded tables
    targetName String Target database name
    targetWarehouse String Custom warehouse location 
    name: sales-dataset
description: Sales database tables
databases:
  - name: sales_db
    includePattern: "fact_.*|dim_.*"
    excludePattern: "test_.*"
    targetName: sales_prod_db
    targetWarehouse: s3a://sales/warehouse

    Job Configuration¶

    Property Type Description
    name String Job name
    description String Job description
    dataStrategy Enum Migration strategy
    dataset String Dataset reference
    configuration String Configuration reference
    leftConnection String Source connection
    rightConnection String Target connection

    Data Strategy Values¶

    Value Description
    SCHEMA_ONLY Schema without data
    LINKED Create linked tables
    SQL Move data via SQL
    EXPORT_IMPORT Use Hive export/import
    HYBRID Automatic strategy selection
    STORAGE_MIGRATION Same-cluster storage move
    COMMON Shared storage
    DUMP DDL extraction only
    ICEBERG_CONVERSION Convert to Iceberg

    Job Options¶

    Property Type Description
    resetToDefaultLocation Boolean Reset table locations
    globalLocationMap Map Namespace mappings
    sqlPartitionLimit Integer Max partitions for SQL
    exportPartitionLimit Integer Max partitions for export 
    name: migration-job
description: Migrate sales to CDP
dataStrategy: HYBRID
dataset: sales-dataset
configuration: migration-config
leftConnection: hdp-cluster
rightConnection: cdp-cluster
options:
  resetToDefaultLocation: false
  sqlPartitionLimit: 100
  exportPartitionLimit: 500

    LLM Configuration¶

    Property Type Description Default
    enabled Boolean Enable LLM features false
    provider Enum LLM provider EMBEDDED_OLLAMA
    model String Model name llama3.1
    temperature Float Creativity (0-1) 0.7
    maxTokens Integer Max response tokens 4096
    endpoint String API endpoint (for remote) null
    apiKey String API key (for cloud) null

    Provider Values¶

    Value Description
    EMBEDDED_OLLAMA Built-in Ollama
    LOCAL_OLLAMA External Ollama
    OPENAI OpenAI API
    ANTHROPIC Anthropic API
    AZURE_OPENAI Azure OpenAI
    AWS_BEDROCK AWS Bedrock
    GOOGLE_AI Google AI 
    llm:
  enabled: true
  provider: LOCAL_OLLAMA
  model: llama3.1
  temperature: 0.7
  maxTokens: 4096
  endpoint: http://localhost:11434

    Environment Variables¶

    Override properties with environment variables:

    Property Environment Variable
    server.port SERVER_PORT
    hms-mirror.rocksdb.path HMS_MIRROR_ROCKSDB_PATH
    hms-mirror.config.output-dir HMS_MIRROR_OUTPUT_DIR 
    export SERVER_PORT=8091
export HMS_MIRROR_ROCKSDB_PATH=/data/hms-mirror/rocksdb
java -jar hms-mirror-server.jar

    Complete Configuration Example¶

    # application.yml

server:
  port: 8090
  servlet:
    context-path: /hms-mirror

hms-mirror:
  rocksdb:
    enabled: true
    path: ${user.home}/.hms-mirror/rocksdb
  config:
    output-dir: ${user.home}/.hms-mirror/reports
    metastore-drivers-dir: ${user.home}/.hms-mirror/aux_libs

logging:
  level:
    com.cloudera.utils.hms: INFO
  file:
    name: ${user.home}/.hms-mirror/logs/hms-mirror.log

spring:
  profiles:
    active: production

    Default Values Reference¶

    Server Defaults¶

    Setting Default
    Port 8090
    Context path /hms-mirror
    RocksDB enabled true
    RocksDB path ~/.hms-mirror/rocksdb

    Migration Defaults¶

    Setting Default
    Copy views false
    Evaluate partition location false
    Sort dynamic partition inserts false
    Auto tune false

    ACID Defaults¶

    Setting Default
    Include ACID tables false
    Downgrade ACID false
    Partition limit 0 (unlimited)
    Bucket threshold 0

    Iceberg Defaults¶

    Setting Default
    Enabled false
    Conversion mode SHADOW
    Version 2
    File format PARQUET

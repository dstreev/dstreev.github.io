    Driver Management¶

    HMS-Mirror requires JDBC drivers to connect to Hive clusters and metastore databases. This section explains how to obtain, install, and configure these drivers.

    Driver Types¶

    HMS-Mirror uses two types of JDBC drivers:

    ┌─────────────────────────────────────────────────────────────────────┐
│                        HMS-Mirror Connections                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌─────────────────────────┐    ┌─────────────────────────────────┐│
│  │    Hive JDBC Drivers    │    │   Metastore JDBC Drivers        ││
│  │    (HiveServer2)        │    │   (Direct DB Access)            ││
│  │                         │    │                                  ││
│  │  • CDP Hive Driver      │    │  • MySQL Connector              ││
│  │  • HDP Hive Driver      │    │  • PostgreSQL Driver            ││
│  │  • CDH Hive Driver      │    │  • Oracle JDBC Driver           ││
│  │  • Apache Hive Driver   │    │  • MariaDB Connector            ││
│  │                         │    │                                  ││
│  │  Location: drivers/     │    │  Location: aux_libs/            ││
│  └─────────────────────────┘    └─────────────────────────────────┘│
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘

    Driver Locations¶

    After installation, drivers are organized in ~/.hms-mirror/:

    ~/.hms-mirror/
├── drivers/              # Hive JDBC drivers (by platform)
│   ├── cdp/              # Cloudera Data Platform drivers
│   ├── hdp/              # Hortonworks Data Platform drivers
│   ├── cdh/              # Cloudera Distribution for Hadoop drivers
│   └── apache/           # Apache Hive open-source drivers
│
└── aux_libs/             # Metastore JDBC drivers
    ├── mysql-connector-j-x.x.x.jar
    ├── postgresql-x.x.x.jar
    ├── mariadb-java-client-x.x.x.jar
    └── ojdbc11-x.x.x.jar

    Quick Setup¶

    Download Scripts¶

    HMS-Mirror includes helper scripts to download drivers:

    # Download Hive JDBC drivers for your platform
~/.hms-mirror/bin/download-hive-drivers.sh

# Download Metastore JDBC drivers
~/.hms-mirror/bin/download-metastore-drivers.sh

    Manual Installation¶

    If automatic download isn't available:

    1. Hive Drivers: Copy from your cluster's client libraries or download from vendor
    2. Metastore Drivers: Download from Maven Central or vendor sites

    Driver Compatibility Matrix¶

    Cluster Type Hive Driver Location Recommended Version
    CDP 7.x drivers/cdp/ Match CDP version
    HDP 3.x drivers/hdp/ HiveJDBC4
    HDP 2.x drivers/hdp/ HiveJDBC41
    CDH 6.x drivers/cdh/ Cloudera JDBC
    CDH 5.x drivers/cdh/ Cloudera JDBC
    Apache Hive drivers/apache/ Match Hive version
    Metastore DB Driver Maven Coordinates
    MySQL MySQL Connector/J mysql:mysql-connector-java
    PostgreSQL PostgreSQL JDBC org.postgresql:postgresql
    MariaDB MariaDB Connector org.mariadb.jdbc:mariadb-java-client
    Oracle Oracle JDBC com.oracle.database.jdbc:ojdbc11

    Common Issues¶

    "No suitable driver found"¶

    • Verify driver JAR is in the correct location
    • Check driver class name in connection configuration
    • Ensure JAR file permissions allow reading

    "Driver version mismatch"¶

    • Match driver version to cluster version
    • Check for conflicting driver versions
    • Review cluster documentation for compatible versions

    "Class not found"¶

    • Driver JAR may be corrupted - re-download
    • Check classpath configuration
    • Verify JAR contains expected classes
